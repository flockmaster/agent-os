# 📖 Antigravity Agent OS 极简使用指南

> **核心理念**：你负责"说想要什么"，Agent 负责"思考怎么做"、"动手执行"和"记住经验"。

---

## 🚀 1. 怎么开始新项目？
**场景**：你要做一个全新的 APP。
1. 把本系统的 `.agent/` 文件夹复制到新项目根目录。
2. 对 Agent 说：**"开始"**。
3. Agent 会说："系统就绪，请吩咐。"

---

## ✨ 2. 怎么开发新功能？
**场景**：你想加一个"用户登录"功能。
1. **你说**："我想加一个微信登录功能。"
2. **Agent**：思考并生成一份《设计方案 (PRD)》，问你："这样行吗？"
3. **你说**："行，Go！"
4. **Agent**：自动写代码、自动测试、自动提交 Git。

---

## 🔥 3. 遇到报错怎么办？
**场景**：程序跑不起来，控制台一大堆红字。
1. **你**：直接把错误日志复制粘贴给 Agent。
2. **Agent**：分析原因 -> 尝试修复 -> 再次运行（它会自动尝试最多 3 次）。

---

## 🧠 4. 这个系统怎么变聪明？
**场景**：你希望 Agent 越用越顺手。
1. **日常**：Agent 每做完一个任务，会自动总结经验（存入记忆库）。
2. **进化**：偶尔输入 `/evolve`，让它整理知识库，变得更聪明。

---

## 🔧 5. 想改 Agent 本身的规则？
**场景**：你觉得 Agent 写的文档太长了，想简化。
1. **你说**：`/meta` 请把 PRD 文档的格式简化一点。
2. **Agent**：修改自己的配置文件（`.agent/` 下的文件），而不动你的业务代码。

---

## 📦 6. 怎么把系统拿给别人用？
**场景**：你朋友觉得这套系统好用，想拿去用。
1. **你说**：`/export`
2. **Agent**：打包一个干净的压缩包（剔除你的私有数据）。

---

## 📂 7. 跨项目迁移/重构 (高级)
**场景**：你有一个旧项目 A，想转换成新项目 B（例如：英语APP -> 中文APP）。
1. **配置**：确保项目 A 中有 `.agent` 系统。
2. **指令**：直接告诉 Agent "我要提取本项目逻辑，重构为一个中文学习软件"。
3. **Agent**：启用 **PRD Crafter Pro**，进行多角色规划和迁移。

---

## ⌨️ 常用命令速查

| 命令 | 作用 | 什么时候用？ |
|-----|-----|-------------|
| `/start` | 启动/恢复 | 刚打开窗口时 |
| `/suspend` | 暂停/存档 | 要下班/休息时 |
| `/status` | 看进度 | 想知道任务做到哪了 |
| `/meta` | 改系统 | 想调整 Agent 行为时 |
| `/rollback` | 后悔药 | 代码改乱了想回退时 |
